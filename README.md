# Scale Vertices to Size (Blender Addon)

This addon allows you to scale selected vertices in Edit Mode to an exact size,
while respecting Blender's current Transform Pivot Point
(Median, Cursor, Active Element, etc.).

## Features
- Scale vertices to an exact target size
- Axis selection (X / Y / Z)
- Displays current bounding box dimensions
- Uses Blender's active transform pivot
- Works in Edit Mode
- Blender 3.1+

## Location
View3D → Sidebar (N) → Edit → Scale Vertices to Size

## Installation
1. Download `scale_vertices_to_size.py`
2. Blender → Edit → Preferences → Add-ons
3. Install… → select the file
4. Enable the addon

## Usage
1. Go to Edit Mode
2. Select vertices
3. Set Transform Pivot Point (Cursor / Median / etc.)
4. Enter Target Size
5. Select axes
6. Click **Scale Vertices to Size**
